Mint an In Game NFT Map

This sample project is intended to illustrate how you can allow players to mint unique tokens that represent a ticket for a game, virtual land, or in game items. These tokens can then be authenticated with the FusedVR Chain Auth SDK within a standalone app to prove ownership of the minted asset. Map Generation powered by Catlike Coding.

Mint an NFT with an ID of your choice below to your wallet and then view the randomly generated map in the Unity Open Source Project.
